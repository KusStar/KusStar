# headless

Rendering [kusstar](../kusstar) with [node-canvas-webgl](https://github.com/akira-cn/node-canvas-webgl)

## Note

Modify [config.json](./config.json) to change output props:
```json
{
  "totalFrame": 120,
  "width": 512,
  "height": 256,
  "frameTime": 16,
  "outputName": "kusstar.gif"
}
```

Or [source](./index.js)
